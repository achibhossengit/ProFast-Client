## Quick Start

1. **Clone the repository**

```bash
git clone https://github.com/achibhossengit/ProFastClient.git
cd ProFastClient
```

2. **Install dependencies**

```bash
npm install
```

3. **Configure Env & Start the development server**

```bash
npm run dev
```

4. **Open in browser**
   By default, the app should be available at:

```
http://localhost:5173
```
